The opendis7-java library provides a complete type-safe Java implementation of both the DIS Protocol version 7 (IEEE 1278.1-2012) and SISO-REF-010 Enumerations specifications, interfaces and objects, all as open source.

  • opendis7-java Distribution Products provides a thorough set of prepared assets for project reuse.
  • edu.nps.moves.dis7.pdus package contains all IEEE DIS Protocol Data Unit (PDU) classes.
  • edu.nps.moves.dis7.utilities package contains network interface and data-conversion utilities.
  • edu.nps.moves.dis7.utilities.stream package contains stream utilities for DIS file saving, conversions, and playback.
  • Experimental DIS XML schema and documentation assist with data-centric processing of DIS streams.

Over 22,000 SISO-REF-010 version 35 enumeration values are provided, autogenerated from the SISO standard.

  • https://www.sisostandards.org/page/ReferenceDocuments (scroll to bottom of page)
  • MV3500 Specification Documents, SISO-REF-010

opendis7-java Distribution Products are available online, including opendis7-full.jar and online Javadoc.

The MV3500 Distributed Simulation Fundamentals course archive provides further examples, presentations and projects produced by the Modeling, Virtual Environments, Simulation (MOVES) Institute at the Naval Postgraduate School (NPS).
